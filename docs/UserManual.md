# User Manual - Student Information System

## Introduction
The Student Information System (SIS) is a Java-based application that helps manage student records.  
It allows users to add, view, search, update, and delete student details through a menu-driven interface.

---

## Running the Program
1. Open IntelliJ IDEA (or any Java IDE).
2. Navigate to `src/StudentInformationSystem.java`.
3. Run the program.
4. The main menu will appear with options.

---

## Menu Options
- **Add New Student**  
  Enter Student ID, Name, Age, Grade, and Contact. Validation ensures age > 0 and grade between 0â€“100.

- **View All Students**  
  Displays all student records in a formatted table.

- **Search Student**  
  Search by Student ID or Name. Shows matching records.

- **Update Student**  
  Modify details of an existing student. Leave a field blank to keep the old value.

- **Delete Student**  
  Remove a student record permanently.

- **Exit**  
  Close the program.

---

## Example Flow
1. Add 3 students.
2. View all students.
3. Search for one student by ID.
4. Update their grade.
5. Delete another student.
6. Exit the program.
